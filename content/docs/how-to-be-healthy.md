# What should I eat?

Mostly vegan, with fish 2-3 times per week[^1].

## Things to avoid

 - Refined sugars
 - Red meat


# When should I eat?

Probably mostly earlier in the day[^2][^3], and within a 8-12 hour
window[^3][^4].


# Should I fast for prolonged periods?

Probably[^1].
